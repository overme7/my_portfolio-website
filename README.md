# My Portfolio Website

A modern personal portfolio website featuring vlogs, photos, and games.

## Tech Stack

- **React** - Frontend framework
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **React Router** - Navigation

## Getting Started

### Prerequisites

1. Install Node.js from [nodejs.org](https://nodejs.org/)

2. Install dependencies:
```bash
npm install
```

3. Start development server:
```bash
npm run dev
```

4. Open your browser to the URL shown (usually http://localhost:5173)

## Customizing Content

### Adding Vlogs
Edit `src/pages/Vlogs.jsx` and add your video URLs to the `vlogs` array.

### Adding Photos
Edit `src/pages/Photos.jsx` and add your image URLs to the `photos` array.

### Adding Games
Edit `src/pages/Games.jsx` and add your game details to the `games` array.

### Personalizing About Page
Edit `src/pages/About.jsx` to update your name, description, and social media links.

## Building for Production

```bash
npm run build
```

The built files will be in the `dist` folder.

## Deployment

You can deploy to:
- [Vercel](https://vercel.com/)
- [Netlify](https://netlify.com/)
